# Règles Web (HTML/CSS/JavaScript)

- Utilise HTML5 sémantique (header, nav, main, footer, article, section)
- Respecte l'accessibilité (ARIA labels, alt texts, contraste)
- Utilise CSS moderne (Flexbox, Grid, custom properties)
- Privilégie JavaScript ES6+ (const/let, arrow functions, async/await)
- Utilise les modules ES6 pour l'organisation du code
- Valide les entrées utilisateur côté client ET serveur
- Optimise les performances (lazy loading, minification)
- Documente les fonctions complexes avec JSDoc
- Respecte les principes de responsive design
