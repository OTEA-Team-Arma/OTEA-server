# Règles Python

- Utilise Python 3.x (syntaxe moderne)
- Respecte PEP 8 pour le style de code
- Utilise les type hints pour améliorer la lisibilité
- Privilégie les f-strings pour le formatage
- Utilise les context managers (with) pour les ressources
- Documente les fonctions avec des docstrings
- Gère les exceptions de manière appropriée
- Privilégie les compréhensions de liste quand approprié
