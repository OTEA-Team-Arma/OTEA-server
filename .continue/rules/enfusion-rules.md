# Règles Enfusion Script (Arma Reforger)

- Respecte les conventions de nommage Bohemia: PascalCase pour les classes, camelCase pour les méthodes
- Utilise les attributs [Attribute()] pour l'exposition dans l'éditeur
- Gère correctement le cycle de vie des entités (Init, Delete)
- Utilise les composants et le système ECS
- Documente les fonctions et classes importantes
- Privilégie les événements et callbacks du moteur
- Vérifie toujours les pointeurs null
- Utilise GetGame() pour accéder aux systèmes globaux
